Properties
  • text-property-contains
  • text-property-contains-case-insensitive
  • text-property-contains-result
  • text-property-contains-operator
First example
  • text-property-contains="first.b.name"
  • text-property-contains-case-insensitive="false"
  • text-property-contains-result="-1"
  • text-property-contains-operator="!="
  • This result equals to: '{{first.a.name}}'.indexOf('{{first.b.name}}') != -1
  • Result: {{first.a.name.indexOf(first.b.name) != -1}}
1st person
  • {{first.a.name}} must contain {{first.b.name}}

Second example
  • text-property-contains="second.b.name"
  • text-property-contains-case-insensitive="true"
  • text-property-contains-result="5"
  • text-property-contains-operator="="
  • This result equals to: '{{second.a.name}}'.toLowerCase().indexOf('{{second.b.name}}'.toLowerCase()) == 5
  • Result: {{second.a.name.toLowerCase().indexOf(second.b.name.toLowerCase()) == 5}}
